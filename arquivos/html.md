## Estrutura Básica



<!DOCTYPE html>

<html lang="pt-BR">

<head>

&nbsp;   <meta charset="UTF-8">

&nbsp;   <title>Meu Site - Página Inicial</title>

</head>

<body>

&nbsp;   <h1>Bem-vindo ao Meu Site</h1>

&nbsp;   <p>Este é o conteúdo principal da página.</p>

</body>

</html>



**<!DOCTYPE>:** Instrução que informa ao navegador qual versão do HTML está sendo utilizada (HTML5).



**<html>:** Elemento raiz que encapsula todo o conteúdo do documento. O atributo lang define o idioma principal.



**<head>:** Contêiner para metadados, links para CSS, scripts e informações não visíveis.



**<title>:** Define o título da página exibido na aba do navegador e usado por mecanismos de busca.



**<body>:** Contém todos os elementos visíveis da página.





## Estrutura Semântica



header: Cabeçalho de uma página ou seção específica.



nav: Define blocos de links de navegação.



main: O conteúdo principal e exclusivo do documento.



section: Representa uma seção genérica de conteúdo.



article: Conteúdo independente (post, notícia, comentário).



footer: Rodapé com informações de autor ou copyright.



#### Conteúdo e Texto



* <h1>...<h6> – Cabeçalhos de diferentes níveis. Sendo h1 o maior e h6 o menor.
* <p> – Define um parágrafo. Usado para textos em geral.

* <br> – Quebra de linha.

* <hr> – Linha horizontal.
* <pre> – Texto pré-formatado.
* <blockquote> – Citação.

* <code> – Importância semântica.
* <em> – Ênfase.
* <small> – Texto menor.
* <sub> / <sup> – Sub e sobrescrito.



Links e Mídia



<a>

Cria hiperlinks para outras páginas ou arquivos.

<img>

Incorpora imagens (requer atributo src).

<video>

Reproduz arquivos de vídeo nativamente.

<canvas>

Área para renderizar gráficos e animações via JS.



Listas

<ul> – Lista não ordenada.

<ol> – Lista ordenada.

<li> – Cada um dos itens de uma lista.

<dl> – Lista de definição.

<dt> – Termo de definição.

<dd> – Descrição do termo de definição.



Tabelas

<table> – Define uma tabela.

<tr> – Define uma linha na tabela.

<td> – Define uma célula de dados na tabela.

<th> – Define uma célula de cabeçalho na tabela.

<thead> – Agrupa o cabeçalho da tabela.

<tbody> – Agrupa o corpo da tabela.

<tfoot> – Agrupa o rodapé da tabela.



Formulários

<form>

Container para coleta de dados do usuário.

<input>

Vários tipos de entrada (text, password, email).

<button>

Botão executável dentro ou fora de forms.

<label>

Legenda acessível para campos de entrada.

<select> - definição

<textarea> definição



Scripts e Metadados

<script> – Insere scripts (JavaScript).

<link> – Link para recursos externos (CSS).

<meta> – Metadados sobre o documento.

<style> – Estilos CSS embutidos.

<noscript> – Conteúdo alternativo para navegadores sem suporte a JavaScript.

<base> – Define a base URL para links relativos.
